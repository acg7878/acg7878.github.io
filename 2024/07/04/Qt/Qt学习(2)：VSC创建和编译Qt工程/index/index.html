Qt学习(2)：VSC创建和编译Qt工程

一、环境变量配置

需要配置4处环境变量，可以参照一下路径自行添加

二、VSC插件

需要安装几个插件：

  1. Cmake和Cmake Tools

  2. Qt tools：拥有启动Qt Creator等功能

  3. Qt Configure：创建Qt工程

插件配置

Qt Configure

Qt Configure: Mingw Path

MinGW 安装路径，需要到 bin 之前的那个目录

Qt Configure: Qt Dir

Qt 安装路径

Qt Configure: Qt Kit Dir

Qt 套件路径

Cmake Tools

Cmake: Cmake Path

填入Qt的cmake执行文件的路径

三、工程创建和编译

工程创建

vscode使用shift+ctrl+p调出主命令框，输入QtConfigure:New Project。按提示创建即可

注意生成的CMakeLists.txt中使用的是Qt5，使用Qt6的小伙伴需要修改成Qt6

编译器路径设置

vscode使用shift+ctrl+p调出主命令框，输入C/C++: 编辑配置(UI)

首先修改一下两项

同时可以添加头文件路径以消除vscode头文件冒红问题

如果无法在GUI模式下填入路径，可以通过json进行修改（ C/C++: 编辑配置(JSON) ）

此外，还需要设置一下工具包。在vscode状态栏的cmake tools处选择Qt的工具包

工程编译运行

至此，即可在状态栏点击构建与运行了，抑或通过快捷键F5进行编译运行
